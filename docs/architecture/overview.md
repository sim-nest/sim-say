# Architecture Overview

SIM is split into public code repositories plus this generated front page.
`sim-say` contains no Rust source and is regenerated from the published
contract files.

Currently published code repositories: sim-agent-net, sim-audio-daw, sim-citizen, sim-codecs, sim-discrete, sim-femm, sim-foundation, sim-kernel, sim-music, sim-numbers, sim-run, sim-runtime, sim-sdk, sim-shape, sim-storage, sim-stream, sim-stream-host, sim-tooling, sim-web.
